Pangeo

About

 Pangeo is a tectonic plate simulator and was my team's submission at the eHacks 2023 hackathon. Pangeo allows you to draw different land shapes on a top down map and set directions for them to move and collide. The project does not accurately represent the movement of tectonic plates, it is only meant as a way to showcase programming skill.

Hall of Fame

About

 This was a project I created for Freeburg Community High School. The app is designed be used to view rewards or honors, such as perfect attendence, state scholars, or state champions, earned by students and teams from the school. Users could navigate through folders of records and also sort them. It also contains a management application for an administrator to add, edit, or remove records. The project is intended to be run on a LAMP stack on the internal network. The management application can then be accessed by any desktop computer and the kiosk app can be accessed on a Chromebook in kiosk mode connected to a interactive display in a hallway.
