Pangeo

About

 Pangeo is a wacky tectonic plate simulator and was my team's submission at the eHacks 2023 hackathon. Pangeo allows the user to draw land masses on a top down map using paint tools along with directions for their corresponding tectonic plates to move and collide. Although the project does not accurately represent real life, it nonetheless can still create cool shapes.
